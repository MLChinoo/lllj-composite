# lllj-composite

**Q1：这是什么？**

A1：适用于游戏《Limelight Lemonade Jam》的立绘合成工具。

**Q2：如何使用？**

A2：使用如下，请自行尝试：

1. 使用[GARbro](https://github.com/MLChinoo/GARbro)搭配[HxNames-LLLJ.lst](https://github.com/MLChinoo/lllj_hxnames)解包fgimage.xp3；
2. 确认每个角色的.stand, .sinfo, .pbd, .tlg文件都在同一目录下，以隐杏珠为例，则目录下应存在这些文件：`杏珠.stand`、`杏珠a.sinfo`、`杏珠a.pbd`、`杏珠a_****.tlg`、`杏珠b.sinfo`、`杏珠b.pbd`、`杏珠b_****.tlg`；
3. 打开此工具，选择解包导出的fgimage目录，再选择Character, Pose, Dress, Addition, Face即可合成立绘；
4. 点击Export可导出单张立绘，旁边Batch Export菜单中可按选定层级范围批量导出。
